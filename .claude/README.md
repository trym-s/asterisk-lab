# Claude Agent Entry Point

Claude agents should treat `../AGENTS.md` as the repo-wide operating rules
and `../docs/specs/` for governing specs.

## Routing

Read `../AGENTS.md` before choosing a skill.

Common mappings:

- SIP endpoint work: `skills/adding-sip-endpoint/SKILL.md`
- SIP registration debugging: `skills/debugging-sip-registration/SKILL.md`
- VM deploys: `skills/deploying-to-vm/SKILL.md`
- Password rotation: `skills/rotating-passwords/SKILL.md`
- Contracted engineering changes: `skills/engineering-mission-playbook/SKILL.md`
- Benchmark or metric work: `skills/optimization-mission-playbook/SKILL.md`

Skills define procedure. Specs and contracts define supported behavior and done.
